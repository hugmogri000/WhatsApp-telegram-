Privacy Policy / 隐私政策

最后更新：2026年3月

1. 概述

本隐私政策适用于 Chrome 扩展程序"Telegram & WhatsApp 实时翻译"（以下简称"本扩展"）。我们非常重视用户隐私，本扩展在设计上遵循最小数据原则。

2. 我们收集的数据

本扩展仅在用户本地浏览器中存储以下信息：

  • 激活码：用于验证用户授权状态，仅存储在本地，不上传至任何服务器
  • 设备标识符：根据浏览器硬件信息生成的匿名标识，用于防止激活码被多设备滥用
  • 语言偏好设置：用户选择的翻译源语言和目标语言
  • 翻译缓存：已翻译的文本结果，存储在本地以提升响应速度，减少重复请求

3. 数据使用方式

  • 用户聊天消息中的文本内容会被发送至翻译服务接口进行翻译，翻译完成后结果直接显示在页面上
  • 我们不存储任何用户的聊天内容
  • 翻译请求仅包含需要翻译的文本、目标语言及授权凭证，不包含任何个人身份信息

4. 数据共享

我们不会将用户数据出售、出租或以任何形式共享给第三方。翻译文本仅发送至本扩展自有的翻译服务接口，不会传递给任何其他组织或个人。

5. 数据安全

  • 所有与翻译服务的通信均通过 HTTPS 加密传输
  • 激活码和设备标识符仅存储在用户本地浏览器的 chrome.storage 中
  • 本扩展不使用任何第三方分析或追踪工具

6. 用户权利

用户可随时通过卸载本扩展来删除所有本地存储的数据。如需手动清除，可在 Chrome 扩展管理页面中清除本扩展的存储数据。

7. 政策变更

如本隐私政策发生变更，我们将在本页面更新内容并注明修改日期。建议用户定期查阅本页面。

1. Overview

This Privacy Policy applies to the Chrome extension "Telegram & WhatsApp Real-time Translator" (hereinafter "the Extension"). We take user privacy seriously and the Extension is designed with a minimal data principle.

2. Data We Collect

The Extension only stores the following information locally in the user's browser:

  • Activation Code: Used to verify user authorization status. Stored locally only, never uploaded to any server.
  • Device Identifier: An anonymous identifier generated from browser hardware information, used to prevent activation codes from being abused across multiple devices.
  • Language Preferences: The source and target languages selected by the user.
  • Translation Cache: Previously translated text results stored locally to improve response speed and reduce repeated requests.

3. How Data Is Used

  • Text content from chat messages is sent to the translation service API for translation, and the result is displayed directly on the page.
  • We do not store any user chat content.
  • Translation requests only contain the text to be translated, the target language, and the authorization credential — no personally identifiable information.

4. Data Sharing

We do not sell, rent, or share user data with any third parties. Translated text is only sent to the Extension's own translation service API and is not passed to any other organization or individual.

5. Data Security

  • All communication with the translation service is encrypted via HTTPS.
  • Activation codes and device identifiers are stored only in the user's local browser via chrome.storage.
  • The Extension does not use any third-party analytics or tracking tools.

6. User Rights

Users can delete all locally stored data at any time by uninstalling the Extension. To manually clear data, users can clear the Extension's storage from the Chrome Extensions management page.

7. Policy Changes

If this Privacy Policy changes, we will update the content on this page and note the revision date. We recommend users check this page periodically.
