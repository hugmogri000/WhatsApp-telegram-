# WA/TG 实时翻译隐私政策 **生效日期 / Last updated：2026-04-14** 感谢您使用 **WA/TG 实时翻译** Chrome 扩展程序（以下简称“本扩展”）。 本隐私政策用于说明：本扩展会处理哪些数据、这些数据如何使用、会与哪些服务发生必要的数据交互，以及用户可以如何理解和管理相关数据。 本扩展的核心用途是： - 为 **Telegram Web** 与 **WhatsApp Web** 提供实时聊天翻译功能 - 支持将接收到的外语消息翻译为目标语言 - 支持将用户输入的文本在发送前翻译为目标语言 - 提供与上述核心功能直接相关的 **微信登录、会员验证、卡密兑换、订单支付和状态恢复** 功能 --- ## 1. 我们可能处理的数据 为了提供上述功能，本扩展可能处理以下数据： ### 1.1 聊天与翻译相关数据 - Telegram / WhatsApp 页面中的待翻译聊天文本 - 用户在输入框中主动提交用于翻译的文本 - 翻译源语言、目标语言等翻译参数 - 为减少重复请求而在浏览器本地保存的翻译缓存结果及必要缓存键 **用途：** 完成实时翻译、发送前翻译，并提升翻译响应速度。 ### 1.2 账号与身份数据 - 微信相关身份标识，例如 OpenID、系统生成的稳定用户ID、昵称（如有） - 插件前端展示用的脱敏身份信息，例如掩码后的 OpenID、用户ID标签 **用途：** 完成微信登录、识别当前账号、绑定或恢复会员权益、展示当前账号状态。 ### 1.3 身份验证与授权数据 - 登录 token - 微信动态码会话信息 - 激活码 / 卡密 - 会员状态、试用状态、到期时间 - 设备标识（deviceId） **用途：** 完成登录校验、会员验证、卡密兑换、账号与设备上下文识别、状态恢复和安全校验。 ### 1.4 订单与支付数据 - 购买套餐 - 订单金额 - 订单状态 - 支付创建时间、支付确认时间 - 区块链交易哈希（tx hash） - 与订单绑定的内部账号标识及设备标识 **用途：** 创建订单、查询支付状态、核验到账、自动开通会员、恢复支付结果、处理售后支持。 ### 1.5 运行与排障数据 - 为保障支付识别、登录验证、服务稳定性和故障排查而生成的必要技术日志或调试信息 **用途：** 安全校验、异常排查、服务稳定性维护。 --- ## 2. 数据如何使用 我们仅将相关数据用于本扩展核心功能，包括但不限于： - 提供 Telegram Web 与 WhatsApp Web 的聊天翻译 - 提供发送前输入翻译 - 完成微信登录与账号识别 - 验证会员、试用和卡密状态 - 创建和查询支付订单 - 核验 TRC20 USDT 支付是否到账 - 在支付成功后自动开通会员或恢复已有权益 - 进行必要的安全校验、故障排查和服务维护 我们**不会**将用户数据用于与本扩展核心功能无关的用途。 --- ## 3. 数据存储位置 ### 3.1 浏览器本地存储 本扩展会在浏览器本地（例如 `chrome.storage.local`）保存与功能直接相关的数据，例如： - 语言设置 - 翻译开关状态 - 登录状态 - 用户身份展示信息 - 会员 / 试用状态 - 卡密 / 授权状态 - 设备标识 - 待支付订单恢复信息 - 本地翻译缓存结果 ### 3.2 服务端存储 为支持登录、账号识别、会员绑定、订单支付、支付核验和状态恢复，本扩展配套服务可能存储以下数据： - 微信身份标识（例如 OpenID、稳定用户ID、昵称等） - 登录 token 与微信动态码会话数据 - 会员与卡密绑定关系 - 订单、支付状态、交易哈希等数据 - 必要的技术日志与故障排查信息 --- ## 4. 第三方服务与必要的数据传输 为了实现本扩展核心功能，相关数据可能在本扩展、自有服务及必要的第三方服务之间进行传输。相关第三方服务包括但不限于： ### 4.1 微信公众号相关服务 用于扫码 / 动态码登录、身份识别和账号确认。 ### 4.2 DeepSeek 翻译服务 用于生成翻译结果。用户提交用于翻译的文本，会通过我们的服务转发给翻译服务提供商进行处理。 ### 4.3 TRONSCAN / 波场链相关查询服务 用于查询 TRC20 USDT 转账记录，以核验支付是否到账。 ### 4.4 Telegram Bot API（如已启用） 如果服务端开启订单通知，支付成功后的订单通知可能会发送给服务运营者，用于订单监控、售后支持和安全审计。通知内容可能包括套餐、金额、交易哈希以及内部用户ID等必要订单信息。 上述第三方服务仅在实现本扩展核心功能所必需时使用。 --- ## 5. 数据共享与出售 - 我们**不会出售**用户数据 - 我们**不会将用户数据用于广告投放** - 我们**不会将用户数据用于与本扩展核心功能无关的用途** 除为实现翻译、登录识别、会员验证、支付核验、订单通知和必要运维支持所必需的情况外，我们不会将用户数据提供给无关第三方。 --- ## 6. 数据保留 我们会在提供登录、会员管理、订单支付、状态恢复、售后支持、安全审计和故障排查所必需的范围内保留相关数据。 其中： -
