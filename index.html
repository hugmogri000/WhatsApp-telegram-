中文

1. 概述

本隐私政策适用于 Chrome 扩展程序“WA/TG 实时翻译”（以下简称“本扩展”）。 我们重视用户隐私，并仅在实现扩展核心功能所必需的范围内处理相关数据。

本扩展的核心用途是为 Telegram Web 与 WhatsApp Web 提供实时聊天翻译功能， 包括接收消息翻译、发送前输入翻译，以及与此直接相关的微信登录、会员验证、卡密兑换、订单支付和状态恢复功能。

2. 我们可能处理的数据

为了提供上述功能，本扩展可能处理以下数据：

  • Telegram / WhatsApp 页面中的待翻译聊天文本
  • 用户在输入框中主动提交用于翻译的文本
  • 翻译源语言、目标语言等翻译参数
  • 为减少重复请求而在浏览器本地保存的翻译缓存结果及必要缓存键
  • 微信相关身份标识，例如 OpenID、稳定用户ID、昵称（如有）
  • 插件前端展示用的脱敏身份信息，例如掩码后的 OpenID、用户ID标签
  • 登录 token、微信动态码会话信息、激活码 / 卡密
  • 会员状态、试用状态、到期时间、设备标识（deviceId）
  • 购买套餐、订单金额、订单状态、支付确认时间、区块链交易哈希（tx hash）
  • 用于安全校验、故障排查和服务稳定性维护的必要技术日志或调试信息

3. 数据如何使用

我们仅将相关数据用于本扩展核心功能，包括但不限于：

  • 提供 Telegram Web 与 WhatsApp Web 的实时聊天翻译
  • 提供发送前输入翻译
  • 完成微信登录与账号识别
  • 验证会员、试用和卡密状态
  • 创建和查询支付订单
  • 核验 TRC20 USDT 支付是否到账
  • 在支付成功后自动开通会员或恢复已有权益
  • 进行必要的安全校验、故障排查和服务维护

我们不会将用户数据用于与本扩展核心功能无关的用途。

4. 数据存储位置

浏览器本地存储：

  • 语言设置
  • 翻译开关状态
  • 登录状态
  • 用户身份展示信息
  • 会员 / 试用状态
  • 卡密 / 授权状态
  • 设备标识
  • 待支付订单恢复信息
  • 本地翻译缓存结果

服务端存储：

  • 微信身份标识（例如 OpenID、稳定用户ID、昵称等）
  • 登录 token 与微信动态码会话数据
  • 会员与卡密绑定关系
  • 订单、支付状态、交易哈希等数据
  • 必要的技术日志与故障排查信息

5. 第三方服务

为了实现本扩展核心功能，我们可能通过自有服务调用第三方服务，包括但不限于：

  • 微信公众号相关服务：用于扫码 / 动态码登录、身份识别和账号确认
  • DeepSeek 翻译服务：经由自有 Cloudflare Worker 转发，用于生成翻译结果
  • TRONSCAN / 波场链相关查询服务：用于核验 USDT 转账是否到账
  • Telegram Bot API（如启用）：用于订单通知、售后支持和安全审计

6. 数据共享与出售

  • 我们不会出售用户数据
  • 我们不会将用户数据用于广告投放
  • 我们不会将用户数据用于与本扩展核心功能无关的用途

除为实现翻译、登录识别、会员验证、支付核验、订单通知和必要运维支持所必需的情况外， 我们不会将用户数据提供给无关第三方。

7. 数据保留

我们会在提供登录、会员管理、订单支付、状态恢复、售后支持、安全审计和故障排查所必需的范围内保留相关数据。

用户卸载扩展或清除扩展本地存储后，本地保存的数据可被移除。

8. 关于远程代码

本扩展不会下载、注入或执行远程 JavaScript、Wasm 或其他可执行代码。

远程服务返回的内容主要为翻译结果、登录状态、会员状态、订单状态等数据，这些内容不会作为代码执行。

10. 政策更新

如果本扩展的功能、数据处理方式、第三方服务或合规要求发生变化，本隐私政策也会相应更新。 请以本页面中的最新版本为准。

English

1. Overview

This Privacy Policy applies to the Chrome extension “WA/TG Real-time Translator” (the “Extension”). We value user privacy and only process data to the extent necessary to provide the Extension’s core functions.

The Extension is designed to provide real-time translation for Telegram Web and WhatsApp Web, including incoming message translation, pre-send input translation, and related WeChat login, membership verification, activation code redemption, order payment, and status recovery features.

2. Data We May Process

  • chat text from Telegram / WhatsApp pages that is needed for translation
  • text entered by the user for pre-send translation
  • translation parameters such as source language and target language
  • locally stored translation cache results and necessary cache keys
  • WeChat-related identifiers such as OpenID, stable user ID, and nickname (if available)
  • masked or display-oriented identity information shown in the extension UI
  • login token, WeChat session / dynamic code information, activation code / license code
  • membership status, trial status, expiration time, and device identifier (deviceId)
  • selected plan, order amount, order status, payment confirmation time, and blockchain transaction hash (tx hash)
  • necessary technical logs or debugging information for security checks and troubleshooting

3. How Data Is Used

  • to provide real-time translation on Telegram Web and WhatsApp Web
  • to translate text before sending
  • to complete WeChat login and account recognition
  • to verify membership, trial, and activation code status
  • to create and check payment orders
  • to verify TRC20 USDT payment status
  • to automatically activate membership or restore existing entitlements
  • to perform necessary security checks, troubleshooting, and service maintenance

We do not use user data for purposes unrelated to the Extension’s core functions.

4. Where Data Is Stored

Browser local storage:

  • language settings
  • translation switch states
  • login state
  • identity display information
  • membership / trial state
  • activation / authorization state
  • device identifier
  • pending order recovery information
  • local translation cache results

Server-side storage:

  • WeChat identity information
  • login token and WeChat session data
  • membership / activation code binding relationships
  • order data, payment status, and transaction hash data
  • necessary technical logs and troubleshooting information

5. Third-Party Services

  • WeChat-related services for login and identity recognition
  • DeepSeek's translation service forwards translations via its own Cloudflare Workers to generate translation results.
  • TRONSCAN / TRON-related query services for verifying USDT transfers
  • Telegram Bot API (if enabled) for order notification and support

6. Data Sharing and Sale

  • We do not sell user data
  • We do not use user data for advertising
  • We do not use user data for purposes unrelated to the Extension’s core functions

7. Remote Code

The Extension does not download, inject, or execute remote JavaScript, Wasm, or other executable code. Remote services return data such as translation results, login state, membership state, and order status, and such data is not executed as code.
